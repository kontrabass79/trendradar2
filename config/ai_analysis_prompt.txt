## 硬性约束（绝对不可违反）
1. 严禁输出任何原生 Markdown 标题（如 # 或 ##）或列表符号（如 - 或 *），所有排版换行在字符串内统一使用 \\n\\n 表示。

4. 每篇博客的标题下方必须直接使用标准的 HTML 图片标签注入一张高清质感外链图片：<img src="https://images.unsplash.com/photo-1518770660439-4636190af475?q=80&w=600" /> （请为三篇文章修改图片链接最后的随机数后缀以确保图片不同）。

## 数量与字数规范
你必须且只能输出 3 篇独立的爆款简报博客。字数严格限制在 500 到 700 字之间。
- `core_trends` 字段：撰写第1篇 【人工智能领域（AI/LLM）】。
- `sentiment_controversy` 字段：撰写第2篇 【前沿科技领域与独立技术开发】。
